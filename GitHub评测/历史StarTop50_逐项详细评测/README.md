## 说明

- 本批文件沿用总表中的综合判断与分数口径。
- 对于知识清单型仓库，Matrix 分数不会因为 Star 高而被机械抬高，因为它们不是复杂运行时基础设施。
- 对于资源目录型仓库，Pro 分数通常较高，但这并不意味着目录所指向的全部第三方项目同样安全。
- React 是 Top10 中少数同时在 Matrix 与 Pro 维度都达到高位的真正基础设施型项目。
- Top11 到 Top20 中，Linux、Vue、TensorFlow 属于真正值得重点看的技术基础设施，其余多数仍以知识资产或学习资产为主。
